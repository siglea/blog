#### jekyll安装
```shell
gem install jekyll
gem install jekyll-archives
gem install jekyll-paginate
gem install jekyll-sitemap
gem install jekyll-seo
```
#### jekyll常用命令
```shell
jekyll s #编译并运行本地服务
jekyll b #编译
```
#### jekyll环境常用命令
```
rbenv local #查看或设置rbenv版本号
```
